连续函数的性质，连续函数的加减乘除后的函数都是连续的。连续函数加减不连续函数得出的函数是不连续的，连续函数和不连续函数的乘除是可连续或者不可连续，如0是连续 0*不连续=连续
所学的基本初等函数在区间内是连续的，在定义域内可能是不连续的。
对于求函数趋近与x0是否连续需要证明x0领域有定义且xo左右极限存在且相等。这就设计到函数极限的证明，函数极限的证明趋近于一个数分析组成是加减还是乘除，得出的是一个确定的数，还是可变的位置数如0/0这类情况，一旦涉及到0/0或者遇到1的无穷次方则需要用到两个重要极限因为对于接近0和1的式子不能简单的当做0和1，因为要看两边趋近的速率如趋近1的时候也要看次方趋近于无穷时的变化率，由此我理解此时应该要运用洛必达分别求导，即求如0/0除号类型的式子分别的变化率也就是速率，这就涉及到了两数相除的极限，无穷则为某是某的高阶，常数为同阶，1为等阶，0为某是某的低阶。对于求极限时候的加减乘除可以分别运用各自极限加减乘除，除法可用等价式子来替换，实际上是各自的速率一样。（趋近于某一个数）而将极限上下分别求导也就是找出各自的等价式子。
